# Meals App

A React application that allows users to search and discover meal recipes using data from an external API. The project demonstrates API integration, dynamic data rendering, search functionality, and state management in React.

## Live Demo

[View Live Demo](https://meals-orpin.vercel.app/)

## Features

- Search for meals by name
- Fetch recipe data from an external API
- Display meal images and details
- Dynamic rendering of search results
- Responsive and user-friendly interface

## Tech Stack

- React
- JavaScript (ES6+)
- CSS
- REST API

## What I Learned

- Fetching data from APIs using `fetch()`
- Managing asynchronous operations in React
- Working with React state using `useState`
- Using `useEffect` for data fetching
- Rendering dynamic content from API responses
- Handling user input and search functionality
